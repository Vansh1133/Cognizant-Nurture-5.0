# Week 2 - Programming Languages

This folder contains the PL/SQL programming exercises completed as part of the Java FSE Training Program.

## Topics Covered

- PL/SQL Control Structures
- Stored Procedures
- Loops
- Conditional Statements
- SQL Queries
- Data Manipulation (UPDATE)
- DBMS_OUTPUT
- Procedure Creation and Execution

---

## Exercises

### Exercise 1 - Control Structures

#### Scenario 1
- Apply a 1% discount to loan interest rates for customers above 60 years of age.

#### Scenario 2
- Promote customers with a balance greater than 10,000 to VIP status.

#### Scenario 3
- Generate reminders for customers whose loans are due within the next 30 days.

---

### Exercise 3 - Stored Procedures

#### Scenario 1
- Process monthly interest for all savings accounts.

#### Scenario 2
- Update employee salaries by applying a bonus percentage based on department.

#### Scenario 3
- Transfer funds between customer accounts after validating sufficient balance.
